# Chrome Web Store screenshot guide

All screenshots must be 1280x800 pixels. These need to be captured on a machine with Chrome and a display (legion2025 or manually).

## Screenshot 1: Instagram context menu

Right-click context menu on an Instagram post showing the "SocialSnag: Download this (HD)" option. Navigate to a public Instagram post with a high-quality image, right-click the image, and capture the full browser window with the context menu visible.

## Screenshot 2: Twitter/X context menu

Right-click context menu on a Twitter/X image showing SocialSnag download options. Navigate to a tweet with an image, right-click it, and capture the browser window with the context menu open.

## Screenshot 3: Popup with download history

The SocialSnag popup open showing download history with a few entries. Click the extension icon in the toolbar to open the popup. Ensure there are 3-5 recent downloads listed from different platforms so the history looks populated.

## Screenshot 4: Options/settings page

The options page showing platform toggles and the advanced mode setting. Open the extension's options page (right-click extension icon > Options). All platform toggles should be enabled, and advanced mode should be visible.

## Screenshot 5: Downloads folder structure

A file manager window showing the Downloads/SocialSnag/ folder organized by platform subfolders (instagram/, twitter/, facebook/). Each subfolder should contain a few example downloaded files to demonstrate the automatic organization.
